## How to use
`1.git clone https://github.com/Suthiphong/PWMModeCCP1`  
`2.cd PWMModeCCP1`  
`3.launch the Chrome browser app "index.html" or Double click "index.html"`

- INPUT (DutyCycle & Frequency & crystal frequency)

- CCPR1L
- T2CON
- TimePrescaler
- Period(T)
- Ton
- Tosc
